# Consumo de energia - Cálculo de uso de watts e valores.
# Este projeto foi criado em Python

Esse projeto foi criado para calcular o consumo de energia mensal em watts e dizendo os custos desses usos.

O programa solicita o **nome do aparelho utilizado**, **potência do aparelho em watts** e o **tempo de uso diário em horas**, calcula o consumo mensal em watts, o valor de 1 kWh e o valor total do uso.

# Formula Utilizada: (**potencia do aparelho** * **uso diário** * **30**) / **1000**
